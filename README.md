# FreeTools.run — 免费、隐私优先的开发者工具箱（Browser-based Dev Tools）

[![Website](https://img.shields.io/badge/Website-freetools.run-0ea5e9)](https://freetools.run)
[![RSS](https://img.shields.io/badge/RSS-rss.xml-10b981)](https://freetools.run/rss.xml)

**FreeTools.run** 提供一组高频、实用的在线工具：**打开即用**、**隐私优先**、**尽量本地处理**（no upload），面向开发者/设计师/运营同学的日常“救火”场景。

- **官网**：[https://freetools.run](https://freetools.run)
- **RSS**：[https://freetools.run/rss.xml](https://freetools.run/rss.xml)

---

## 你会用到的场景

- **日志/接口调试**：JSON/SQL/XML 格式化、cURL 转代码、JWT 解码
- **安全与兼容**：Base64、URL 编解码、时间戳转换
- **图片与资产**：HEIC 转 JPG/PNG/WEBP、SVG 转 PNG/ICO、生成 favicon 包
- **文档与协作**：Markdown / Mermaid 在线预览与编辑、文本 diff

## 隐私说明（Privacy）

- **优先本地处理**：尽量在浏览器中完成转换/格式化/生成等操作
- **不要求登录**：大多数工具无需账号即可使用

> 温馨提示：部分“查询类”工具（例如 IP 查询）天然需要网络请求；请以页面提示为准。
